# Trader Sentiment Analysis — HyperLiquid

> **Objective:** Analyze how market sentiment (Fear / Greed Index) affects trader behaviour and performance on HyperLiquid, and produce actionable trading strategies backed by data.

---

## Project Structure

```
trader-sentiment-analysis/
├── note.ipynb                 # Main analysis notebook (Parts A–D)
├── dashboard.py               # Interactive Streamlit dashboard
├── fear_greed_index.csv       # Fear & Greed Index data
├── historical_data.csv        # HyperLiquid trade-level data
├── requirements.txt           # Python dependencies
├── outputs/                   # Saved charts & tables (generated by notebook)
├── SUMMARY.md                 # One-page methodology + insights write-up
└── README.md                  # This file
```

## Setup

### Prerequisites

- **Python 3.9+** (tested on 3.9–3.12)
- `pip` package manager

### 1. Clone / download the project

```bash
cd trader-sentiment-analysis
```

### 2. Create a virtual environment (recommended)

```bash
python -m venv venv
source venv/bin/activate        # macOS / Linux
# venv\Scripts\activate         # Windows
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

The `requirements.txt` includes:

| Package | Purpose |
|---------|---------|
| pandas | Data manipulation |
| numpy | Numerical operations |
| matplotlib | Static charts |
| seaborn | Statistical visualizations |
| plotly | Interactive charts |
| scikit-learn | ML models & clustering |
| scipy | Statistical tests |
| streamlit | Interactive dashboard |

## How to Run

### Jupyter Notebook (main analysis)

Open in **VS Code** or **JupyterLab**:

```bash
# VS Code (recommended)
code note.ipynb

# OR Jupyter Lab
jupyter lab note.ipynb
```

Run all cells sequentially (**Kernel → Restart & Run All**).

The notebook is structured in four parts:

| Part | Description | Cells |
|------|-------------|-------|
| **A** | Data preparation — loading, cleaning, timestamp alignment, metric creation | 1–25 |
| **B** | Analysis — performance by sentiment, behavioral shifts, trader segmentation, heatmaps, insights | 26–49 |
| **C** | Actionable output — segment × sentiment tables, strategy proposals, decision matrix | 50–57 |
| **D (Bonus)** | Predictive model (Random Forest / Gradient Boosting), KMeans clustering, archetype analysis | 58–70 |

### Save Output Charts & Tables

After running the full notebook, run the dedicated **"Save Outputs"** cell (added at the end) to export all charts and CSV tables to the `outputs/` folder.

### Streamlit Dashboard

```bash
streamlit run dashboard.py
```

Opens at `http://localhost:8501` with four tabs:

1. **Overview** — Dataset stats, sentiment distribution, PnL summary
2. **Sentiment Impact** — Fear vs. Greed performance comparison
3. **Trader Segments** — Segment-level drill-down (frequency, consistency, size)
4. **Predictions** — ML model results & feature importance
